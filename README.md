# «Розробка веб-орієнтованої інформаційної системи для автоматизації діяльності закладу швидкого харчування»

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра
Відокремленого структурного підрозділу
«Полтавський політехнічний фаховий коледж Національного технічного університету
«Харківський політехнічний інститут»» (ВСП «ППФК НТУ «ХПІ»), 45 група.

Автор – **Семенко Іван**.
Керівник роботи – **Ковальова Наталія Володимирівна**.

## Завдання до дипломної роботи
Розробка веб-орієнтованої інформаційної системи для автоматизації діяльності
закладу швидкого харчування.

## Що реалізовано

- головна сторінка з hero-блоком
- інтерактивне меню з категоріями
- пошук та фільтрація
- картка товару з вибором розміру та додатків
- динамічний перерахунок ціни
- кошик з редагуванням кількості
- оформлення замовлення
- історія замовлень
- демо-адмін-панель для керування меню
- світла / темна тема
- локальне збереження даних через localStorage
- повністю український інтерфейс

## Використані технології

- React 18
- TypeScript
- Vite
- React Router DOM
- Lucide React
- CSS3 без зовнішніх UI-фреймворків
- localStorage

## Запуск

```bash
npm install
npm run dev
```

Після запуску відкрий у браузері адресу, яку покаже Vite (зазвичай `http://localhost:5173`).

## Продакшн-збірка

```bash
npm run build
npm run preview
```

## Особливості демо-версії

- backend не використовується
- меню, кошик, тема та історія замовлень зберігаються у браузері
- адмін-панель працює локально без авторизації
- є кнопка скидання тестових даних в адмін-панелі

## Формат редагування в адмін-панелі

Для полів **Розміри** та **Додатки** використовується формат:

```text
Стандарт:0, Подвійний:55
Сир:25, Бекон:30
```
